Quick start

Easy to test in three moves.

Keep the copy short, let the examples do the work, and point people into the docs before they get lost.

Metadata

Pull a random asset for a category and get a stable JSON shape back.

curl "https://api.neko-love.com/api/v4/neko"

Direct image

Open the returned path directly or embed it in a client app.

GET /api/v4/images/neko/04.webp

Filter endpoint

Test a filtered image with a remote public source and tune `pixel_size` when needed.

curl "https://api.neko-love.com/api/v4/filters/pixelate?image=https://example.com/cat.png&pixel_size=12"
If you want live “Try it out” from GitHub Pages, remember to allow your Pages domain with CORS_ALLOW_ORIGINS on the API side.
